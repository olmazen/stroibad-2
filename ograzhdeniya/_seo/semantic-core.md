## 0. Позиционирование раздела

**Гибрид B2C-премиум + B2B-объекты.** EGOE — завод полного цикла (Балаково), поэтому раздел ограждений закрывает два спроса одной страницей:

- **Премиум-частник** — «архитектура участка»: модульные ограждения с лазерной перфорацией (серия FOREST), скрытая LED-подсветка, дизайн под дом. Высокий чек, эмоциональная подача, калькулятор «посчитай свой проект».
- **Объекты / застройщики / госзаказ** — территории ЖК, школы, детсады, парки, дизайн-код, 44-ФЗ, паспорта изделий, единый RAL со всем благоустройством двора.

Общий якорь доверия: **«производим и монтируем сами»** — реальный завод, а не посредник. Это же — главное УТП против агрегаторов-перекупов в выдаче.

## 3. Мета-теги (эталон для главной раздела)

```
<title>Металлические ограждения от производителя — изготовление на заказ | EGOE</title>
<meta name="description" content="Металлические ограждения от производителя: секционные, 3D-панели, декоративные с лазерной резкой и LED-подсветкой, штакетник, перила. Изготовление по размерам объекта, любой RAL, монтаж и отгрузка по России. Смета за 1 день.">
H1: Металлические ограждения от производителя
```
Правила:
- В `<title>` — главный ключ + «от производителя» + бренд. До ~60 симв. видимой части.
- В description — перечисление типов (ловит длинный хвост) + УТП (по размерам, RAL, монтаж, смета за 1 день). До ~160 симв.
- H1 один на странице. H2 — по кластерам (виды, декор, LED, калькулятор, объекты, FAQ).

## 6. Гэпы конкурентов (за что цепляемся в выдаче)

Проанализировано по выдаче (masterovit, zaborkin, egoza, grandline, prof-zabory, lepse, kovkahome, zaborum):

1. **Никто не совмещает премиум-эмоцию (LED, лазерный узор, «архитектура участка») с заводской B2B-надёжностью (44-ФЗ, паспорта, RAL, свой цех).** — Наш уникальный разворот.
2. **Живой калькулятор с визуализацией.** У конкурентов — «цена от N ₽» и форма. У нас — ползунок → видно ограждение → видно смету. Сильный поведенческий сигнал (время на странице, глубина).
3. **Реальная лазерная перфорация с художественными сюжетами (FOREST).** У большинства — типовая ковка или голая 3D-сетка.
4. **Скрытая LED-подсветка как часть ограждения** (заменяет садовое освещение) — почти не представлено у прямых конкурентов.
5. **Завод-производитель, а не перекуп** — прямая цена, сроки от 7 дней, отгрузка по РФ.

## 7. Технические SEO-требования

- Canonical на `https://www.egoe-life.ru/ograzhdeniya/` (уже стоит).
- Один H1, логичная иерархия H2/H3 по кластерам.
- `alt` у всех изображений — с ключами по смыслу («декоративное ограждение с лазерной резкой, серия FOREST», «секционное ограждение территории»).
- Скорость: webp (уже), lazy-loading ниже первого экрана, критичный CSS/hero — eager.
- Мобильная адаптивность калькулятора (ползунки, визуализатор).
- Обновить `sitemap.xml` при добавлении подстраниц.
- Микроразметка цен (Offer) — «от N ₽ за секцию».
- Тексты — не переспам: ключ в H1/H2/первом абзаце/alt/ссылках, дальше — естественно.
