Вопросы

Частые вопросы

Какая высота перил по нормам?

Для лестниц общественных зданий — 900 мм, для пандусов МГН — двухуровневые поручни 700 и 900 мм. Закладываем нормативы в чертёж.

Сталь или нержавейка?

Для улицы экономичнее сталь с порошковой окраской; нержавейка — для интерьеров и представительских входных групп. Посчитаем оба варианта.

Нужен ли замер?

Для типовых маршей хватает чертежей. Для сложных лестниц используем опросный лист с размерами или выезд замерщика.
